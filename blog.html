What is the Document Object Model (DOM)?

Document Object Model (DOM) is a programming interface for web documents where HTML document represents as a tree of objects which allowing developers to dynamically access, manipulate, and update content, structure, and styles of a webpage using languages like JavaScript.With the DOM, we can transform a static webpage into an interactive experience, allowing us to manipulate almost every aspect of the page in real-time.

How do you select an element from the DOM?

There are couple of way to select an element from DOM. We can select an element from the DOM using methods like document.getElementById(), document.querySelector(), or document.getElementsByClassName(). Personally I prefer to select an element by getElementById().

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where event listener is assigned to a parent element for managing events triggered by its child elements. It's useful because it improves performance by reducing the number of event listeners, simplifies code and works for dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using methods like setAttribute(), getAttribute(), or by directly accessing properties like element.id. For styles, modify the style property, e.g., element.style.color = "blue". And if we use CSS framework like tailwind we can access the classlist and add remove classes of css properties.
